# 🌐 MyUEMF Social Network

![Laravel](https://img.shields.io/badge/Laravel--red) ![PHP](https://img.shields.io/badge/PHP--blue) ![TailwindCSS](https://img.shields.io/badge/TailwindCSS--teal) ![MySQL](https://img.shields.io/badge/MySQL--brightgreen)

## Project Overview

This project focuses on developing a **social networking platform** designed specifically for the **UEMF community** (students, professors, and staff). The platform allows users to share posts, engage in direct messaging, receive notifications, and filter posts by categories such as announcements and events. 

---

## 🎯 Project Objectives

- Develop a platform to facilitate communication and social interaction within the UEMF community.
- Provide features for users to create and manage posts, stories, messages, and notifications.
- Securely store user data and posts using **MySQL** databases.
- Implement user-friendly interfaces for smooth navigation and interaction.

---

## 🚀 Features

1. **User Authentication** 🔐:
   - Secure login and sign-up functionality with email and password.
   
2. **Posts and Stories** 📝:
   - Users can create and share stories, posts, and status updates.
   - Posts can include text, images, videos, or polls.

3. **Messaging System** 💬:
   - Real-time messaging between users.
   - Notification system for new messages.

4. **Friend Suggestions** 🤝:
   - Suggest friends based on profile visits and interactions.
   - Ability to add or invite friends.

5. **Notifications** 🔔:
   - Users receive notifications when mentioned in posts or comments.

---

## 🛠️ Technologies Used

- **Laravel**: PHP framework for back-end development, ensuring robust security.
- **JavaScript**: Used to enhance interactive elements on the front-end.
- **HTML & CSS**: Structure and style the application's user interface.
- **Tailwind CSS**: Utility-first CSS framework for a clean, modern design.
- **MySQL**: Database management system for storing user data and interactions.
- **PhpMyAdmin**: Tool for managing MySQL databases.
- **PHPStorm**: IDE used for developing and managing the project.

---

## Development Methodology

1. **Front-End Development**: Created user-friendly interfaces for login, posts, messaging, and notifications using **HTML**, **CSS**, and **Tailwind CSS**.
2. **Back-End Development**: Built a secure back-end using **Laravel**, ensuring protection against security threats like SQL injections.
3. **Database Management**: Stored and managed user data, posts, messages, and notifications in **MySQL** using **phpMyAdmin**.
4. **Testing and Debugging**: Regularly tested and debugged the application to ensure smooth functionality and secure data handling.
